# Blood Donor Admin Portal

> Agent context for this repository. See [AGENTS.md](./AGENTS.md) for coding guidelines.

This is a Next.js 16 application using Supabase for auth and database, deployed on Vercel.

## Environment Variables

Copy `.env.example` to `.env` and fill in your values. See `.env.example` for all required variables.

## Development

```bash
npm install
npm run dev
```
